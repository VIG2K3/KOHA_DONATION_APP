1. Download the ZIP file from GitHub.
2. Unzip the file.
3. Open the file in Android Studio and run the code.
4. If it shows any error messages, try switching the Gradle JDK version to ms-17.
    1. To do that, go to settings and select SDK Manager
    2. Under (Build, Execution, Deployment), select Build Tools and under that select Gradle. (Can even search for Gradle in the search bar inside SDK Manager)
    3. In Gradle JDK, choose ms-17 (Microsoft OpenJDK, version 17)
    4. Click Apply, Clean Project and Rebuild Gradle.
